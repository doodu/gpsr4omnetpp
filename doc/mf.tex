\section{使用mf}
\subsection{安装mf}
需要OMNeT++ version 3.0a8或者更高版本
\enumerate
\item tar -xzf mobility-fw-version.tgz
\item Add  your/path/to/mobility-fw-version/lib
      to your LD_LIBRARY_PATH variable
\item cd mobility-fw-version/
\item make install
\item  make
\endenumerate

\subsection{创建自己的网络}
为了更简单的创建自己的网络可以使用现有的模板，模板文件在tmplate/目录下
\enumerate
\item 创建一个自己的目录如yourNetwork,拷贝下面几个文件到这个目录YourNetwork.ned,YourHost.ned，Makefile.gen如果使用了自己的模块，要把模块文件也拷贝过来
\item 把Makefile.gen中的环境变量MOBFW设成MF的根目录
\item 修改YourHost.ned文件
\item 运行 make -f Makefile.gen 生成一个 Makefile，然后再运行make
\endenumerate
